 Codveda Java Projects

This repository contains Java projects developed during the Codveda internship program.
The projects were created to practice core Java concepts such as OOP, file handling, loops, methods, conditionals, recursion, and basic application logic.

Projects Included

 1. Bank Account
A simple Java project that simulates bank account operations such as:

* Deposit
* Withdraw
* Balance checking

2. Banking Application
A console-based banking management system with basic banking functionalities and user interaction.

3. Employee Management System
A Java application used to manage employee information and operations such as:

* Add employee
* Update employee
* Display employee data

4. File Processing
A project focused on Java file handling:

* Reading files
* Writing files
* Processing text content

5. Calculator
A basic calculator application implementing arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

 6. Factorial Program
A Java program that calculates factorial values using loops and/or recursion.

7. Guess Number Game
A simple number guessing game where the user tries to guess a randomly generated number.

Technologies Used
* Java
* OOP Concepts
* File Handling
* Console-based Applications

These projects were completed as part of the Codveda Internship Program.
